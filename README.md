### リスト形式

#### 丸点

文頭に`*`、`+`、`-`をいれて記入する。上下に改行入れる。

例

```
* list1
* list2
* list3
```

見た目

* list1
* list2
* list3

#### 数字リスト

```
1. item
1. item
1. item
```

1. item
1. item
1. item
